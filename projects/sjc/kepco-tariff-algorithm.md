# KEPCO 요금 계산 알고리즘

## 개요

한전(KEPCO) 전기요금 체계를 알고리즘화한 계산 공식 정리

---

## 공통 계산 공식

### 최종 청구금액 구조

```
최종청구금액 = 전기요금 + 부가가치세 + 전력산업기반기금

전기요금 = 기본요금 + 전력량요금 + 기후환경요금 + 연료비조정액
```

### 공통 부가 항목

| 항목 | 계산식 | 비고 |
|------|--------|------|
| 부가가치세 | 전기요금 × 10% | |
| 전력산업기반기금 | 전기요금 × 3.7% | 10원 미만 절사 |
| 기후환경요금 | 총사용량 × 9원/kWh | |
| 연료비조정액 | 총사용량 × 5원/kWh | |

### 계절 구분

- **하계**: 7월 ~ 8월
- **기타계절**: 3~6월, 9~2월
- **동계/하계 구분 적용**: 일반용 시간대별 요금

---

## 일반용(갑)II 고압A

### 특징

- 시간대별 요금 (경부하/중간부하/최대부하)
- 계절별 단가 차등 적용

### 기본요금

```
기본요금 = 계약전력(kW) × 8,230원/kW
```

### 전력량요금

```
전력량요금 = (경부하사용량 × 경부하단가)
           + (중간부하사용량 × 중간부하단가)
           + (최대부하사용량 × 최대부하단가)
           → 원미만 절사
```

### 단가표 (원/kWh)

| 구분 | 경부하 | 중간부하 | 최대부하 |
|------|--------|----------|----------|
| 여름 | 73.0 | 114.5 | 178.7 |
| 봄·가을 | 73.0 | 85.3 | 114.5 |
| 겨울 | 92.8 | 123.2 | 138.0 |

### 계산 예시

```
입력:
- 계약전력: 250 kW
- 경부하: 150 kWh, 중간부하: 250 kWh, 최대부하: 350 kWh
- 계절: 겨울철

계산:
기본요금 = 250 × 8,230 = 2,057,500원
전력량요금 = (150 × 92.8) + (250 × 123.2) + (350 × 138.0)
          = 13,920 + 30,800 + 48,300 = 93,020원
총사용량 = 150 + 250 + 350 = 750 kWh
기후환경요금 = 750 × 9 = 6,750원
연료비조정액 = 750 × 5 = 3,750원
전기요금 = 2,057,500 + 93,020 + 6,750 + 3,750 = 2,161,020원
부가가치세 = 2,161,020 × 0.1 = 216,102원
전력산업기반기금 = 2,161,020 × 0.037 = 79,957 → 79,950원
최종청구금액 = 2,161,020 + 216,102 + 79,950 = 2,457,072원 → 2,457,070원
```

## 일반용(을)II 고압A

### 특징

- 일반용(갑)보다 높은 단가
- 동일한 시간대별 요금 구조

### 기본요금

```
기본요금 = 계약전력(kW) × 8,320원/kW
```

### 단가표 (원/kWh)

| 구분 | 경부하 | 중간부하 | 최대부하 |
|------|--------|----------|----------|
| 여름 | 74.4 | 128.1 | 197.9 |
| 봄·가을 | 74.4 | 92.8 | 128.1 |
| 겨울 | 94.3 | 140.4 | 197.9 |

### 계산 예시 (동일 조건)

```
기본요금 = 250 × 8,320 = 2,080,000원
전력량요금 = (150 × 94.3) + (250 × 140.4) + (350 × 197.9)
          = 14,145 + 35,100 + 69,265 = 118,510원
기후환경요금 = 750 × 9 = 6,750원
연료비조정액 = 750 × 5 = 3,750원
전기요금 = 2,080,000 + 118,510 + 6,750 + 3,750 = 2,209,010원
부가가치세 = 2,209,010 × 0.1 = 220,901원
전력산업기반기금 = 2,209,010 × 0.037 = 81,733 → 81,730원
최종청구금액 = 2,209,010 + 220,901 + 81,730 = 2,511,641원 → 2,511,640원
```

## 주택용(저압)

### 특징

- 3구간 누진제
- 하계 구간 확대 적용

### 기본요금 (사용량 구간별)

| 사용량 | 기본요금 |
|--------|----------|
| 200kWh 이하 | 910원 |
| 201~400kWh | 1,600원 |
| 400kWh 초과 | 7,300원 |

### 전력량요금

```
전력량요금 = (1구간 사용량 × 1구간 단가)
           + (2구간 사용량 × 2구간 단가)
           + (3구간 사용량 × 3구간 단가)
           → 원미만 절사
```

### 구간 분배 (기타계절: 3~6월, 9~2월)

| 구간 | 범위 | 단가(원/kWh) |
|------|------|--------------|
| 1구간 | 0~200kWh | 120.0 |
| 2구간 | 201~400kWh | 214.6 |
| 3구간 | 400kWh 초과 | 307.3 |

### 구간 분배 (하계: 7~8월) - 구간 확대

| 구간 | 범위 | 단가(원/kWh) |
|------|------|--------------|
| 1구간 | 0~300kWh | 120.0 |
| 2구간 | 301~450kWh | 214.6 |
| 3구간 | 450kWh 초과 | 307.3 |

### 계산 예시 (기타계절, 350kWh)

```
구간 분배:
- 1구간: 200kWh
- 2구간: 150kWh (350 - 200)
- 3구간: 0kWh

기본요금 = 1,600원 (2단계)
전력량요금 = (200 × 120) + (150 × 214.6) = 24,000 + 32,190 = 56,190원
기후환경요금 = 350 × 9 = 3,150원
연료비조정액 = 350 × 5 = 1,750원
전기요금 = 1,600 + 56,190 + 3,150 + 1,750 = 62,690원
부가가치세 = 62,690 × 0.1 = 6,269원
전력산업기반기금 = 62,690 × 0.037 = 2,319 → 2,310원
최종청구금액 = 62,690 + 6,269 + 2,310 = 71,269원 → 71,260원
```

### 계산 예시 (하계, 350kWh)

```
구간 분배 (하계 확대 적용):
- 1구간: 300kWh (확대됨)
- 2구간: 50kWh (350 - 300)
- 3구간: 0kWh

기본요금 = 1,600원 (2단계)
전력량요금 = (300 × 120) + (50 × 214.6) = 36,000 + 10,730 = 46,730원
기후환경요금 = 350 × 9 = 3,150원
연료비조정액 = 350 × 5 = 1,750원
전기요금 = 1,600 + 46,730 + 3,150 + 1,750 = 53,230원
부가가치세 = 53,230 × 0.1 = 5,323원
전력산업기반기금 = 53,230 × 0.037 = 1,969 → 1,960원
최종청구금액 = 53,230 + 5,323 + 1,960 = 60,513원 → 60,510원
```

## 심야전력(을)II

### 특징

- 심야/주간 시간대 구분
- 심야 시간대 저렴한 단가
- 주로 축열/축냉 설비에 적용

### 기본요금

```
기본요금 = 계약전력(kW) × 4,520원/kW
```

### 전력량요금

```
전력량요금 = (심야사용량 × 심야단가) + (주간사용량 × 주간단가)
           → 원미만 절사
```

### 단가표 (원/kWh)

| 구분 | 단가 | 시간대 |
|------|------|--------|
| 심야 | 71.8 | 23:00 ~ 09:00 |
| 주간 | 113.9 | 09:00 ~ 23:00 |

### 계산 예시

```
입력:
- 계약전력: 100 kW
- 심야사용량: 500 kWh
- 주간사용량: 200 kWh

계산:
기본요금 = 100 × 4,520 = 452,000원
전력량요금 = (500 × 71.8) + (200 × 113.9)
          = 35,900 + 22,780 = 58,680원
총사용량 = 500 + 200 = 700 kWh
기후환경요금 = 700 × 9 = 6,300원
연료비조정액 = 700 × 5 = 3,500원
전기요금 = 452,000 + 58,680 + 6,300 + 3,500 = 520,480원
부가가치세 = 520,480 × 0.1 = 52,048원
전력산업기반기금 = 520,480 × 0.037 = 19,257 → 19,250원
최종청구금액 = 520,480 + 52,048 + 19,250 = 591,778원 → 591,770원
```
