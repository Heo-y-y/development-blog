## What I Did

### 1. 에너지 데이터 시각화 대시보드
- 전주시 용도별 전기/가스 소비 현황 (도넛 차트)
- 태양광 보급·발전용량 추이 시각화 (바+라인 차트)
- 온실가스 배출현황 분석 (스택드 컬럼 차트)

### 2. 탄소 절감 캠페인 대시보드
- "전주시민이 에너지 관리로 심은 나무" 시각화
- 탄소 절감량을 나무 그루 수로 환산하여 시민 친화적 표현
- 탄소흡수량, 전기집중량, 전기절감량 실시간 표시

### 3. 실시간 전력수급 현황
- 전력거래소(KPX) 공공데이터 API 연동
- 5분 단위 발전원별 전력수급 데이터 시각화
- SWR 기반 실시간 데이터 갱신

### 4. 지역별 에너지 분석
- 행정구역(구/동)별 전기·가스 소비량 분석
- 카카오맵 API 연동 태양광 설비 위치 표시
