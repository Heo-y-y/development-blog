## 💪 What I Did

### 1. IoT 게이트웨이 연동
- 다양한 에너지 장치와의 데이터 수집 인터페이스 구현
- 실시간 데이터 수집 파이프라인 구축

### 2. 실시간 모니터링 대시보드 API
- 에너지 사용량 실시간 조회 API 개발
- 대시보드용 집계 데이터 제공

### 3. 에너지 사용량 분석 리포트
- 기간별 에너지 사용량 분석 기능
- 리포트 생성 및 다운로드 기능

### 4. 데이터 수집 스케줄링
- Spring Scheduler 기반 주기적 데이터 수집
- 장애 복구 및 재시도 로직 구현
