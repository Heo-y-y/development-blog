# 리액트 생성 및 전체 구조 생성하기

### vite를 이용한 리액트 생성

```jsx
npm init vite
```

### 필요한 패키지 설치

```jsx
npm install dayjs html-react-parser @reduxjs/toolkit
react-quill uuid react-toastify styled-components
react-icons react-router-dom react-redux
```

```jsx
npm install -D @types/styled-components @types/uuid
```
