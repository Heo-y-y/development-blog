## 풀이

```sql
SELECT *
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```

전체 데이터를 출력하기 위해 `SELECT은 *`로 설정했다.

그리고 `ORDER BY ANIMAL_ID`로 동물 아이디 순으로 가져오게 설정했다.
