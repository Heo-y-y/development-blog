2. `npm start` 또는 `npx serve -s build`으로 실행하여 해당 링크로 확인

## 주요 컴포넌트 및 폴더 구성

### 각 기능에 필요한 Component

- 글 목록 보기 기능(리스트 형태)
    - PostList, PostListItem
- 글 보기 기능
    - Post
- 댓글 보기 기능
    - CommentList, CommentListItem
- 글 작성 기능
    - PostWrite
- 댓글 작성 기능
    - CommentWrite
